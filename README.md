# 🏥 Hospital Management System

> A console-based Hospital Management System built with **pure Python** — no external libraries required.

---

## 📌 About the Project

This project is a fully functional **Hospital Management System** that runs in the terminal. It was built to practice core Python concepts like dictionaries, loops, conditionals, and user input handling. The system simulates real-world hospital workflows including patient registration, doctor management, appointment booking, and billing.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔐 Login System | Secure admin login before accessing any module |
| 🧑‍⚕️ Patient Management | Register, view, search, update, and delete patient records |
| 👨‍⚕️ Doctor Management | Register and view doctors with specialization and consultation fee |
| 📅 Appointment Booking | Book appointments by linking patients with doctors |
| 💰 Billing System | Auto-calculates total bill (doctor fee + medicine fee) |

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Libraries:** None (pure Python only)
- **Concepts Used:** Dictionaries, Loops, Conditionals, User Input Handling, CRUD Operations

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/hospital-management-system.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd hospital-management-system
   ```

3. **Run the script**
   ```bash
   python hospital.py
   ```
